# 🧠 OrderGenie AI – Voice Activated Order Management System

An AI-driven voice-based order management system that allows users to create, modify, track, and cancel orders using natural voice commands instead of manual input.

---

## 🚀 Overview

Traditional order management systems rely on manual interactions, which can slow down operations and introduce errors.  
OrderGenie AI solves this problem by enabling **hands-free, voice-based interaction** using AI and NLP.

---

## 🎯 Key Features

- 🎤 Voice-based order creation (e.g., “Create order for two pizzas”)  
- 🔄 Modify, track, and cancel orders using natural language  
- 🧠 NLP-based intent recognition using LLMs  
- ⚡ Real-time order processing and updates  
- 🛒 Smart order suggestions and validation  
- ♿ Accessible interaction for elderly and visually impaired users  

---

## 🏗️ Tech Stack

- **Frontend:** React, Tailwind CSS  
- **Backend:** Flask (Python)  
- **Database:** MongoDB Atlas  
- **AI/NLP:** Ollama (LLM), OpenAI Whisper (Speech-to-Text)  
- **Other:** REST APIs, JSON  

---

## 🔄 System Workflow

Voice Input → Speech-to-Text (Whisper) → NLP Processing (LLM) → API Execution → Database Update → Response Output  

---

## 💡 Use Cases

- Retail POS systems
